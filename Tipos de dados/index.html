Variáveis em JS

6 primitivos:

  • Boolean (true/false)
  • null (dado nulo)
  • undefined (uma propriedade superior cujo valor é indefinido)
  • Número - Number (inteiro ou racional [casas decimais])
  • String (textos)
  • Symbol (um tipo de dado cuja as instâncias são únicas e imutáveis)
e Object.
